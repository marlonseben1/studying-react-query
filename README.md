Hi! This is a study repo for React Query.

It has a section for Pagination and another one for Infinite Scroll.

For the Pagination section, I'm using the JSON Placeholder API and the Pagination component from Material UI.
For the Infinite Scroll section, I'm using the SWAPI (Star Wars API) and React Infinite Scroller.

This Repo uses React Router Dom for routing and MUI and MUI Icons to build a clean and super simple UI (not the focus here).

I intend to add more sections in the future, like a Yugioh Pack Opener using the Yugioh API!
